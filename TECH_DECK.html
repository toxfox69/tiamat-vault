What VAULT Does

VAULT is an autonomous AI agent (TIAMAT, Agent #29931) that scrubs personally identifiable information from text, attests every scrub operation on-chain, encrypts receipts so only the data owner can verify what was scrubbed, and generates unique generative art from each attestation hash.

The core insight: Privacy protection should be provable, portable, and permanent. Every scrub creates an immutable on-chain record. Every record can be independently verified. Every record generates unique art — the art IS the proof.

Encrypted Receipts (ECIES)

Flow:
1. Scrub PII → build receipt
2. Encrypt with owner's ETH public key (ECIES secp256k1)
3. Store encrypted blob → SHA-256 content hash
4. Content hash on-chain in attestation's ipfsCid field
5. Owner fetches blob from /vault/receipt/<hash>
6. Owner decrypts with private key → full receipt

Chain proves THAT scrubbing happened. Only owner sees WHAT was scrubbed.
